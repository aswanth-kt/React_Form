# 🚀 React Forms – Login & Signup

## 📌 Project Overview

This project is part of my ongoing journey to strengthen my React development skills, focused on building **Login and Signup forms** with proper state management, validation, and user experience improvements.

The goal was to understand how forms work in React and how to structure them in a clean and scalable way.

---

## 🎯 Objective

* Learn React form handling using `useState`
* Implement form validation logic
* Improve UI/UX with real-time feedback
* Build reusable and maintainable components

---

## ✨ Features

### 🔐 Authentication Forms

* Login page
* Signup page

### ✅ Form Validation

* Required field validation
* Email format validation
* Password match validation
* Real-time error handling

### 👁️ Password Visibility Toggle

* Show/Hide password functionality
* Independent control for each password field

### ⚠️ Error Handling

* Field-level error messages
* Conditional rendering based on user interaction (`touched` state)

### ⚡ UX Improvements

* Live validation (on blur)
* Disabled button during submission
* Clean and minimal UI using Tailwind CSS

---

## 🧠 Concepts Covered

* Controlled components in React
* State management using `useState`
* Form validation patterns
* Conditional rendering
* Reusable component design
* Handling user interactions (`onChange`, `onBlur`)

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/aswanth-kt/React_Form.git
cd React_Form
```

### 2. Install dependencies

```
npm install
```

### 3. Run the app

```
npm run dev
```
